Getting Started

The fastest way to get started with writing Ema app is as follows:

  1. Install Nix (see platform-specific notes here)
  2. Enable Flakes
  3. Clone the template repository locally
  4. Run bin/run and access the site at http://localhost:9001

Next, we will walk over how to develop a basic site using Ema.
